# 在线演示

> 由于文档站点的限制，完整的 3D 模型查看器演示需要在单独的 Demo 项目中运行。

## 运行 Demo

```bash
# 安装依赖
pnpm install

# 运行 Demo 项目
pnpm dev:demo
```

## Demo 功能

Demo 项目展示了以下功能：

1. **多模型切换** - 通过下拉列表切换不同的 3D 模型
2. **完整交互** - 所有交互功能均可测试
3. **功能展示** - 侧边栏列出所有核心功能

## 模型列表

Demo 中包含以下模型：

- 🧀 奶酪模型（默认）
- 🍦 巧克力冰淇淋模型
- 🍔 汉堡模型

## 使用说明

1. 启动 Demo 后，默认显示奶酪模型
2. 使用鼠标滚轮缩放模型
3. 左键拖动旋转视角
4. 右键拖动平移视图
5. 使用右上角控制面板调整各种参数
6. 使用下拉列表切换不同模型

## 注意事项

- 模型文件较大，首次加载可能需要一些时间
- 建议使用现代浏览器（Chrome、Firefox、Safari、Edge）
- 移动端支持手势操作
